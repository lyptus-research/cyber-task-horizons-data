# Human Study Data

Expert completion and estimation data from 484
sessions across 11 cybersecurity professionals.

## Files

| File | Description |
|------|-------------|
| **completions.csv** | 174 expert task attempts (pass, fail, censored) with timing |
| **estimations.csv** | 310 solution-visible time estimates with confidence |
| **expert_survey.csv** | Post-study survey (experience levels, qualitative feedback) |
| **DATA_DICTIONARY.md** | Column definitions, notes, and quick-start code |
| human_snapshot.json | Raw API snapshot (same data, nested JSON format) |

Start with `DATA_DICTIONARY.md` for column definitions and usage examples.
